Installing Scorch

To click the links to 'view/play' the music, you need a browser extension called 'Scorch'. If you have not already installed it then click one of these links to download and install Scorch for your particular operating system:

  • MAC
  • Windows

Windows Installation

If you use Windows then, before you can install Scorch, you will need to enable "Internet Explorer" mode in your Microsoft Edge browser. You can do this as follows:

  • Open Microsoft Edge (click Start and type 'Edge', then press Enter).
  • In the address bar, type edge://settings/defaultbrowser. Press Enter.
  • Slide the “Allow sites to be reloaded in Internet Explorer” toggle to ON.
  • Restart Microsoft Edge.

Now, you can install Scorch using the download link above. After installing Scorch, browse to the hymnal website. Now switch to Internet Explorer mode as follows:

  • Click the three dots in the upper right corner of the browser window.
  • Select “Reload in Internet Explorer Mode”.

Now the 'view/play' hymnal links should work in the Microsoft Edge browser in Windows.
