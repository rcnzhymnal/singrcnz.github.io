Projection of Songs for Public Singing

We plan to make powerpoint slides of all the songs in this hymnbook project. Below is copyright information that you need to know to project these songs for public singing.

Note: this page only applies to projection for public singing. Permissions have already been obtained for full public use of the print publication Sing to the Lord (provisional).

Psalm Projection Slides

Almost all of the Psalm slides may be used by the Reformed Churches of New Zealand free of charge: we have either been given permission or, for a few, we have paid for it. You can download the Psalm slides here. But note that we have not been able to obtain permission to project Psalms 57b or 139 at all.

However, a number of these require special notes to be placed on the slides as we have done. If you make up your own slides, then you will also have to place these notes. Here is the full list of songs and special notes.
